### Featured Projects
 
**[E-commerce Data Platform](https://github.com/abiralpokhrel-learns/ecommerce-data-platform)** &nbsp;·&nbsp; *Python · PostgreSQL · dbt · Airflow · Spark · Kafka · MinIO · Docker · Metabase*
 
End-to-end analytical platform for ~100K Brazilian e-commerce orders. Star schema modeled in dbt with surrogate keys (fixed a 30% row-count inflation), daily ingestion orchestrated in Airflow 2.8 with dynamic task mapping, Kafka + Spark Structured Streaming for live event aggregation, dbt and Great Expectations tests enforced via GitHub Actions, findings surfaced in a Metabase dashboard. Runs locally with one `docker-compose up`.
 
**[Open Source Trends Intelligence Platform (ghtrends)](https://github.com/abiralpokhrel-learns/ghtrends)** &nbsp;·&nbsp; [Live demo](https://ghtrends.streamlit.app/) &nbsp;·&nbsp; *AWS Lambda · S3 · Athena · Glue · dbt · Terraform · Streamlit*
 
Serverless AWS data platform that ingests every public GitHub event hourly, models trends with dbt-core on a Glue + Athena lakehouse, and runs semantic search over the top 500 trending repos using sentence-transformers. 100% Terraform-provisioned (Lambda, EventBridge, Glue, S3, IAM, S3+DynamoDB remote state). Total operating cost: under $2/month.
